# Telegram English Bot

Телеграм-бот для изучения английского языка, который отправляет слово дня каждый день.

## Функционал

- `/start` - Начать работу с ботом и получить слово дня
- `/word` - Получить текущее слово дня
- `/help` - Показать справку по командам
- `/test` - Проверить работу бота и посмотреть статус
- Обработка обычных сообщений - бот приветствует пользователя

## Инструкция по запуску

### 1. Установка зависимостей

Убедитесь, что у вас установлен Python 3.8 или выше. Затем установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

Или установите зависимости вручную:

```bash
pip install python-telegram-bot==20.7 python-dotenv==1.0.0
```

### 2. Получение токена бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Введите имя вашего бота (например: "English Word Bot")
   - Введите username вашего бота (должен заканчиваться на `bot`, например: `english_word_bot`)
4. После создания бота BotFather отправит вам токен (выглядит примерно так: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. Сохраните этот токен - он понадобится на следующем шаге

### 3. Настройка переменных окружения

Создайте файл `.env` в корневой директории проекта (там же, где находится `bot.py`).

Содержимое файла `.env` (скопируйте из `.env.example` и заполните):
```
BOT_TOKEN=ваш_токен_бота_здесь
DATABASE_URL=postgresql://USER:PASSWORD@host:5432/DATABASE
```

**Важно:** Замените значения на реальные. Для лексики используется PostgreSQL (например [alwaysdata](https://www.alwaysdata.com)): укажите `DATABASE_URL` или `PGHOST`, `PGPORT`, `PGDATABASE`, `PGUSER`, `PGPASSWORD`.

Пример:
```
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
DATABASE_URL=postgresql://user:пароль@postgresql-сервер.alwaysdata.net:5432/имя_базы
```

### 4. Запуск бота

Запустите основной файл бота:

```bash
python bot.py
```

Если команда `python` не работает, попробуйте:

```bash
python3 bot.py
```

или

```bash
py bot.py
```

### 5. Проверка работы

После запуска вы должны увидеть в консоли сообщение:
```
INFO - Бот запущен!
INFO - Первая отправка слова дня будет в [дата и время]
```

Теперь откройте Telegram, найдите вашего бота по username и отправьте команду `/start`.

## Как проверить команды

1. `/start` - Подписка на ежедневные слова и получение первого слова дня
2. `/word` - Получить текущее слово дня в любое время
3. `/help` - Увидеть список всех доступных команд
4. `/test` - Проверить статус работы бота, увидеть ваш ID и информацию о подписке
5. Напишите любое сообщение - бот поприветствует вас

## Особенности работы

- **Автоматическая отправка слова дня**: Бот отправляет слово дня всем подписанным пользователям каждый день в **9:00 утра** (по времени сервера)
- **База слов**: Бот содержит 30 различных английских слов с переводами и примерами использования
- **Выбор слова**: Слово дня выбирается на основе текущей даты, поэтому в один день все пользователи получат одно и то же слово

## Устранение проблем

### Ошибка "BOT_TOKEN не найден"
- Убедитесь, что файл `.env` создан в правильной директории (рядом с `bot.py`)
- Проверьте, что токен указан правильно (без пробелов, кавычек и т.д.)
- Убедитесь, что вы установили библиотеку `python-dotenv`: `pip install python-dotenv`

### Ошибка при установке зависимостей
- Обновите pip: `pip install --upgrade pip`
- Убедитесь, что используете Python 3.8 или выше: `python --version`

### Бот не отвечает
- Проверьте, что бот запущен (нет ошибок в консоли)
- Убедитесь, что токен правильный (можно создать нового бота через BotFather)
- Проверьте интернет-соединение

## Остановка бота

Для остановки бота нажмите `Ctrl+C` в терминале, где запущен бот.

## Автопополнение слов по API

При добавлении слова можно написать только **слово** — перевод, транскрипция (IPA) и пример подставятся автоматически:
- [Free Dictionary API](https://dictionaryapi.dev/) — определения и транскрипция;
- [MyMemory](https://mymemory.translated.net/) — перевод на русский (без ключа, лимит по запросам).

Формат вручную: `слово — перевод` или `слово / перевод`.

## Запуск в Docker

```bash
# Соберите образ и запустите (нужен файл .env с BOT_TOKEN и DATABASE_URL)
docker-compose up -d --build
```

Остановка: `docker-compose down`.
